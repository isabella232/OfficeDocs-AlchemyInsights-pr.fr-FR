# <a name="activate-visio"></a>Activer Visio

Pour qu’un utilisateur puisse installer et activer l’application de bureau Visio incluse dans son Visio Online Plan 2, l’utilisateur doit avoir reçu une [licence Visio.](https://docs.microsoft.com/microsoft-365/admin/add-users/add-users)

**Notez** Visio Online Plan 1 n’inclut pas de version de bureau de Visio.

Pour activer Visio :

1. Installez Visio à partir de votre [portail Mon](https://portal.office.com/account#installs) compte s’il n’est pas déjà installé. Pour obtenir de l’aide, consultez [Installer Visio](https://support.office.com/article/f98f21e3-aa02-4827-9167-ddab5b025710?wt.mc_id=OfficeAdm_ClientDIA_Alchemy1800).
2. Une fois l’installation terminée, ouvrez l’application Visio. Visio est susceptible de s’activer automatiquement et vous pouvez commencer à l’utiliser immédiatement. Si ce n’est pas le cas, vous pouvez être invité à vous connecter. Veillez à vous connecter à l’aide du même compte professionnel ou scolaire auquel une licence Visio a été attribuée. Suivez les invites pour continuer l’activation. 

Pour les problèmes d’activation, exécutez [l’outil de dépannage](https://aka.ms/SARA-OfficeActivation-Alchemy) de l’activation sur l’ordinateur concerné.
