# <a name="how-to-customize-permissions-for-a-sharepoint-list-or-library"></a>Comment personnaliser les autorisations d’une SharePoint ou d’une bibliothèque

En tant qu’administrateur ou propriétaire d’une bibliothèque, d’une liste ou d’une enquête, vous pouvez modifier les autorisations pour autoriser un groupe de personnes à accéder aux données dont elles ont besoin tout en limitant les autres.

[Personnaliser les autorisations pour une SharePoint ou une bibliothèque](https://support.office.com/article/customize-permissions-for-a-sharepoint-list-or-library-02d770f3-59eb-4910-a608-5f84cc297782)
