# <a name="microsoft-stream-external-sharing"></a>Partage externe Microsoft Stream

Microsoft Stream ne prend pas encore en charge les scénarios de partage externe pour les invités ou les personnes extérieures à votre organisation. Nous savons que ces fonctionnalités sont très nécessaires et que nous travaillons sur des plans pour créer ces fonctionnalités.

- [Partage externe dans Stream](https://docs.microsoft.com/stream/portal-share-video#external-sharing)

- [Partager vos vidéos dans Microsoft Stream](https://docs.microsoft.com/stream/portal-share-video)
