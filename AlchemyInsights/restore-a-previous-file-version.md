# <a name="restore-a-previous-file-version"></a>Restaurer une version de fichier précédente

Avec l’historique des versions en ligne, vous pouvez voir et restaurer des versions antérieures de vos fichiers dans OneDrive ou SharePoint. L’historique des versions fonctionne avec tous les types de fichiers, y compris les fichiers PDF, les fichiers CAD, les photos et les vidéos. Pour restaurer une version précédente du fichier :
  
1. Ouvrez la SharePoint de documents ou l OneDrive emplacement.
    
2. Cliquez sur la gauche du fichier pour le sélectionner, puis cliquez sur Historique **des versions.** Si vous ne voyez pas l’historique des versions dans la barre de commandes, cliquez sur Plus ( **...** ). 
    
3. Cliquez sur la version que vous souhaitez afficher, ou cliquez sur Plus ( **...** ) à droite d’une version, puis cliquez sur **Restaurer**.
