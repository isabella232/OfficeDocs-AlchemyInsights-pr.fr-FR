# <a name="troubleshoot-issues-with-apple-mdm-push-certificate-in-microsoft-intune"></a>Résoudre les problèmes avec le certificat Push Apple MDM dans Microsoft Intune

Examinez les ressources répertoriées ci-dessous pour résoudre votre problème maintenant.
  
1. Examinez [ce document](https://docs.microsoft.com/intune/apple-mdm-push-certificate-get) pour configurer ou renouveler le certificat Push Apple MDM.

2. Utilisez [le portail de dépannage Intune pour](https://devicemanagement.microsoft.com/#blade/Microsoft_Intune_DeviceSettings/TroubleshootBlade) diagnostiquer et résoudre les échecs d’inscription courants. Pour [plus d’informations,](https://docs.microsoft.com/intune/help-desk-operators) examinez ce document.

3. Consultez ces documents pour obtenir une liste des erreurs courantes qui empêchent l’inscription et les solutions à chacune d’elles : [Guide de résolution des problèmes](https://support.microsoft.com/help/4039809/troubleshooting-ios-device-enrollment-in-intune) et [Résolution des problèmes liés aux documents](https://docs.microsoft.com/troubleshoot/mem/intune/troubleshoot-device-enrollment-in-intune).
