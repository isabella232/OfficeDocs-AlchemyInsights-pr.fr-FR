# <a name="get-help-with-windows-10-desktop"></a>Obtenir de l’aide sur l’ordinateur de bureau Windows 10

- Pour créer un raccourci bureau, consultez [Épingler des applications et dossiers sur le bureau ou la barre des tâches](https://support.microsoft.com/windows/pin-apps-and-folders-to-the-desktop-or-taskbar-f3c749fb-e298-4cf1-adda-7fd635df6bb0).

- Pour agrandir les icônes ou le texte dans Windows 10, consultez[Modifier la taille des icônes de bureau](https://support.microsoft.com/windows/change-the-size-of-your-desktop-icons-85a9d341-2a4f-3d96-c796-ae116a187211).

- Pour afficher les icônes du bureau dans Windows 10 consultez, [Afficher des icônes de bureau dans Windows 10](https://support.microsoft.com/windows/show-desktop-icons-in-windows-10-c13270f0-3812-c71d-f27e-29aa32588b20).
