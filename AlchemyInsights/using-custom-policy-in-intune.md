# <a name="using-custom-policy-in-intune"></a>Utilisation d’une stratégie personnalisée dans Intune

Intune dispose de nombreux paramètres intégrés pour les appareils Android, iOS, macOS et Windows mobiles. Vous pouvez également créer un profil personnalisé, puis déployer ces paramètres personnalisés sur vos appareils à l’aide d’Intune. Les profils personnalisés sont conçus pour ajouter des paramètres et des fonctionnalités d’appareil qui ne sont pas intégrés à Intune.

[Paramètres personnalisés pour les appareils Android dans Intune](https://docs.microsoft.com/intune/custom-settings-android)

[Paramètres personnalisés pour les appareils Enterprise Android dans Intune](https://docs.microsoft.com/intune/custom-settings-android-for-work)

[Paramètres personnalisés pour les appareils iOS dans Intune](https://docs.microsoft.com/intune/custom-settings-ios)

[Paramètres personnalisés pour les appareils macOS dans Intune](https://docs.microsoft.com/intune/custom-settings-macos)

[Paramètres personnalisés pour Windows 10 appareils mobiles dans Intune](https://docs.microsoft.com/intune/custom-settings-windows-10)

[Paramètres personnalisés pour Windows Holographic for Business appareils mobiles dans Intune](https://docs.microsoft.com/intune/custom-settings-windows-holographic)
