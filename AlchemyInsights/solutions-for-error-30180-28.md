# <a name="solutions-for-error-30180-28"></a>Solutions pour l’erreur 30180-28

Vous pouvez voir [l’erreur 30180-28](https://support.office.com/article/47ae453b-677c-412f-9a21-6766555ff4de?wt.mc_id=Alchemy_ClientDIA) pendant une mise à jour Office si vous avez une connexion Internet médiocre. Here are some things you can try to fix the problem.
  
- Redémarrez votre ordinateur, puis mettez [à jour Office](https://support.office.com/article/2ab296f3-7f03-43a2-8e50-46de917611c5?wt.mc_id=Alchemy_ClientDIA).

- Suivez les étapes ci-dessus, mais utilisez une connexion câblé lors de la mise à jour Office.
