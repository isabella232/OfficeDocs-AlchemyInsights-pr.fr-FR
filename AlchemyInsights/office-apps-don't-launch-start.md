# <a name="microsoft-365-apps-crash-stop-responding-or-dont-launch"></a>Microsoft 365 applications se crashent, cessent de répondre ou ne se lancent pas

Pour corriger Microsoft 365 applications se bloquent ou se bloquent, essayez les options suivantes :

- [Examiner les problèmes éventuels avec les Office](https://support.office.com/article/powerpoint-isn-t-responding-hangs-or-freezes-652ede6e-e3d2-449a-a07f-8c800dfb948d#bkmk_addins)
- Vérifier si un [logiciel antivirus est en conflit](https://support.office.com/article/powerpoint-isn-t-responding-hangs-or-freezes-652ede6e-e3d2-449a-a07f-8c800dfb948d?ocmsassetID#bkmk_conflict) avec Microsoft 365 applications
- Vérifier ou [installer les dernières mises à jour pour Office](https://support.office.com/article/update-office-and-your-computer-with-microsoft-update-2ab296f3-7f03-43a2-8e50-46de917611c5)
- [Réparer les Office](https://support.office.com/article/repair-an-office-application-7821d4b6-7c1d-4205-aa0e-a6b40c5bb88b) à l’aide du Panneau de commande
- [Désinstaller](https://support.office.com/article/uninstall-office-from-a-pc-9dd49b83-264a-477a-8fcc-2fdf5dbf61d8) et réinstaller Office
