# <a name="delete-a-folder-in-outlookcom"></a>Supprimer un dossier dans Outlook.com

1. Dans le volet gauche, cliquez avec le bouton droit sur le dossier ou le sous-dossier que vous souhaitez supprimer et sélectionnez **Supprimer le dossier.**
2. Sélectionnez **OK** pour confirmer.

Pour plus d’informations, voir [Working with message folders in Outlook.com](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a?wt.mc_id=Office_Outlook_com_Alchemy).
